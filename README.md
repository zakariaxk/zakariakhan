# Zakaria Khan

This portfolio is a software artifact as much as a visual one: a personal system for presenting engineering taste, technical range, and product thinking through an interactive interface.

## Stack

- `Next.js`: application framework and rendering layer for structuring the site as a fast, production-grade React app.
- `React`: component model for building the interface as composable interactive systems instead of static pages.
- `TypeScript`: static type layer for keeping UI logic, content structures, and component contracts explicit.
- `Framer Motion`: motion engine for transitions, scroll-linked state, and responsive interaction choreography.
- `React Three Fiber`: React renderer for WebGL scenes, used to treat 3D graphics as part of the component architecture.
- `Drei`: higher-level helpers on top of React Three Fiber for scene composition and reusable 3D primitives.
- `Three.js`: low-level graphics foundation that powers the spatial rendering and visual atmosphere behind the experience.

## What It Is

A portfolio built from a computer science perspective: interface engineering, typed frontend architecture, motion systems, and real-time graphics working together to communicate how I think and build.
